Mis Proyectos


E-commerce

sistema de eCommerce en PHP para la venta de celulares. Los usuarios pueden navegar, buscar, filtrar productos, realizar compras y pagar en línea. El administrador gestiona inventarios, pedidos y usuarios. Utiliza MySQL para la base de datos, PHP para la lógica de negocio, y CSS y JavaScript para el diseño y la interacción en el frontend.

Gestor administrativo para un refugio animal

Sistema de escritorio en Java desarrollado con Apache NetBeans para gestionar refugios de animales. Permite administrar adopciones, registrar mascotas, voluntarios y adoptantes, y generar reportes. Utiliza JDBC con MySQL y Swing para la interfaz gráfica.

Crowdfunding

plataforma de Crowdfunding en Java Web usando JSP para la interfaz y JDBC para la conexión con MySQL. Los usuarios pueden crear y donar a campañas, con una interfaz interactiva mejorada con CSS y JavaScript.
